API de gestion des réservations, realisé par Houssam-Eddine Imhiouach

Une application FastAPI pour gérer les clients, les chambres et les réservations.

Lancement du serveur:

1. Installer les dépendances :
   pip install fastapi uvicorn sqlalchemy pymysql

2. Démarrer le serveur :
   Dans le dossier du projet (ou se trouve main.py), lancez :
   uvicorn main:app --reload

Le serveur sera disponible à l’adresse :
- http://127.0.0.1:8000/docs (Documentation interactive)
